Τα ζώδια της ημέρας

Φίλη Παρθένε μην έχεις νεύρα, γιατί θα το μετανιώσεις. Άκου μας.

Tweet

Το Cosmo.gr σε συνεργασία με το astro-live.gr γράφουν κάθε μέρα τι προβλέπει το ζώδιό σας.

Κριός

Αν και θέλετε να διασκεδάσετε σήμερα και να είστε κομμάτι της παρέας, δεν θέλετε να χαθείτε μέσα στο πλήθος. Το γεγονός πως ο πλανητης ερμης σας δινει έξτρα κοφτερή γλώσσα, δεν σημαίνει πως μπορείτε να τη χρησιμοποιήσετε αλόγιστα πάνω στους άλλους. Δυστυχώς, η συμπεριφορά αυτή θα οδηγήσει σε προβλήματα, εκτός αν σκεφτείτε προσεκτικά πριν μιλήσετε. Απαλύνετε το μήνυμά σας έτσι ώστε οι άλλοι να μην χρειαστεί να καλύψουν τα αυτά τους.

Ταύρος

Δεν σας χαροποιεί όταν γίνεστε στόχος για τον σαρκασμό των άλλων, αλλά η υπεράσπιση του εαυτού σας ταιριάζει στην πεισματάρική σας φύση. Ωστόσο, ίσως να μην είναι εύκολο για εσάς να ανταποκριθείτε στη συγκεκριμένη κατάσταση σήμερα. Μερικές φορές είναι πιο εύκολο να κρύψετε το θυμό παρά να αντιμετωπίσετε τη ρίζα του προβλήματος. Να θυμάστε πως δεν είναι πράξη δειλίας. Ο περιορισμός αυτός είναι σημάδι σοφίας. Αλλιώς μπορεί να χάσετε τον έλεγχο και να δυσκολέψετε τα πράγματα ακόμα περισσότερο.

Δίδυμοι

Πρέπει να είστε σε επαγρύπνηση, αλλιώς η έντασή σας θα ξεκινήσει να ακούγεται υπερβολικά επιθετική στους άλλους. Ίσως πιστέψετε πως είστε οι μόνοι λογικοί σήμερα, μόνο για να συνειδητοποιήσετε πως οι ιδέες σας δεν μπορούν να γίνουν πραγματικότητα. Αντί να εξηγείτε την κάθε λεπτομέρεια, μιλήστε καλύτερα και λιγότερο. Δεν χρειάζεται να χρησιμοποιήσετε πολλές λέξεις για να περάσετε το μήνυμά σας. Όμως, ίσως χρειαστεί να περιμένετε λίγο πριν το κατανοήσουν ολοκληρωτικά.

Καρκίνος

Θα τους εκπλήξετε όλους σήμερα με την ικανότητά σας να παλέψετε για τις επιλογές σας. Θα έχετε τη θέληση να ακούσετε, αλλά μόλις πάρετε τις προσφορές, η αντίδρασή σας θα είναι ισχυρή. Ακόμα και αν ξέρετε τις απόψεις των άλλων, προσέξτε να μην αλλοιωθεί η δύναμη του μηνύματός σας. Αν προσπαθήσετε να καλύψετε υπερβολικά άγνωστο έδαφος με τη μία, δεν θα φτάσετε πολύ μακριά.

Λέων

Δεν είναι δύσκολο για εσάς να εκφράσετε τις δημιουργικές σας ιδέες, αλλά σήμερα υπάρχει τόση πολλή θετική ενέργεια γύρω σας που δεν θα ξέρετε πότε να σταματήσετε. Η αυτοπεποίθηση θα σας βάλει στον πειρασμό να γίνετε τολμηροί, με αποτέλεσμα να προσβάλετε ακόμα και τους πιο ισχυρούς σας συμμάχους. Τα πράγματα θα πάνε μια χαρά αν θυμηθείτε πως ο διας μεγεθύνει ό, τι και αν λέτε, και κάνει τα λόγια σας να ακούγονται πιο σημαντικά από ότι είναι. Προσέξτε να μην σας θεωρήσουν αλαζονικούς.

Παρθένος

Οι αλληλεπιδράσεις σας θα είναι συναισθηματικά ευμετάβλητες σήμερα, καθώς φτάνετε σε ένα σημείο καμπής όσον αφορά μία σχέση. Οι αρνητικές σκέψεις κάποιου θα αρχίσουν να μειώνονται. Όμως μην κάνετε το λάθος και θεωρήσετε πως λιγότερος θόρυβος σημαίνει λιγότερη ένταση. Μην επιτρέψετε στον εαυτό σας να νιώσει θυμό, διότι θα πείτε πράγματα που δεν τα εννοείτε. Κάντε πίσω μέχρι να βρείτε ξανά την ισορροπία σας. Μόνο έτσι θα γλιτώσετε τις εντάσεις.

Ζυγός

Θέλετε να ευχαριστείτε τους ανθρώπους τόσο πολύ που μερικές φορές ανησυχείτε για την ευτυχία των άλλων αφήνοντας τη δική σας στην άκρη. Όμως, η σεληνη στο παρορμητικό ζωδιο του κριου, πέφτει πάνω στον πλανητη ουρανο και στην στοργικη αφροδιτη σημερα, σηματοδοτώντας πως μπορεί να ξεφουρνίσετε κάτι μέσα από τη καρδιά σας χωρίς να σκεφτείτε τον αντίκτυπο. Βάλτε κάποια όρια για να εξασφαλίσετε την υγεία των σχέσεών σας στο μέλλον.

Σκορπιός

Επιτέλους έχετε το κίνητρο να ξεκινήσετε νέα πρότζεκτ, με τόσους πλανήτες να επισκέπτονται τον 6 οικο σας. Δυστυχώς, θα δυσκολευτείτε να ολοκληρώσετε αυτά που θα αρχίσετε, αν δεν διατηρήσετε την ταχύτητα και την πορεία σας. Ίσως χάσετε το δρόμο σας αν επιβραδύνετε, για αυτό ρωτήστε τους φίλους και τους συγγενείς αν μπορούν να σας βοηθήσουν ή αν γίνεται να βγουν από το δρόμο σας. Τα πάντα είναι δυνατά αν τα βάλετε στο μυαλό σας.

Τοξότης

Ακόμα και αν το μήνυμά σας ακούγεται σημαντικό, μην περιμένετε από τους άλλους να παρατήσουν αυτό που κάνουν και να σας ακολουθήσουν αμέσως. Αν και οι φίλοι γνωρίζουν πόσο φιλοσοφικοί είστε, χάνουν την υπομονή τους όταν ενθουσιάζεστε για κάτι που είναι σημαντικό μόνο για εσάς. Οι ιδέες έρχονται με απίστευτα ταχύτητα, αλλά είναι δύσκολο να καταλάβετε ποιες αξίζει να τις κυνηγήσετε.

Αιγόκερως

Είναι ωραίο να κρατάτε την εξουσία, αλλά μπορεί να το παρακάνετε πολύ εύκολα. Η ιδέα του να αποδείξετε την δύναμή σας θα είναι τόσο προκλητική, που ίσως χάσετε τις αυθεντικές σας προθέσεις. Ωστόσο, η επιτυχία δεν έρχεται από τον απόλυτο έλεγχο. Το να φτάσετε τους στόχους σας εξαρτάται από την καινοτομία και την συνεργασία, όχι την ιεραρχία και την παράδοση.

Υδροχόος

Η θετική σκέψη είναι η πιο ισχυρή δύναμη που δίνει κίνητρο, αλλά ίσως το παρακάνετε σήμερα. Η τυφλή αισιοδοξία δεν είναι χρήσιμη, καθώς αν οι γύρω σας πιστεύουν πως δεν μπορείτε να ξεχωρίσετε την πραγματικότητα από τη φαντασία, υπάρχει πιθανότητα να χάσουν την πίστη που τρέφουν στο πρόσωπό σας. Υποστηρίξτε το σχεδιο σας με το να είστε όσο το δυνατόν πιο ρεαλιστικοί, ακόμα και στις λεπτομέρειες. Ολοι θελουν να ακουσουν καλά νέα, αλλά είναι πιο σημαντικό να πείτε την αλήθεια.

Ιχθύες

Ανυπομονείτε να δημιουργήσετε νέες στρατηγικές για να αυξήσετε το εισόδημά σας, όσο η σεληνη στο ζωδιο του κριου ταξιδεύει στον 2 οικο σας. Θα έχετε πολλές λαμπρές ιδέες για να βελτιώσετε τα οικονομικά σας, αλλά δεν ξέρετε ποια από όλες θα δουλέψει. Να θυμάστε πως η μακροχρόνια παρουσία του ποσειδωνα στο ζωδιο σας ίσως ρουφήξει όλη σας την αυτοπεποίθηση, για αυτό μην διστάσετε να ζητήσετε την υποστήριξη των άλλων. Βασιστείτε στις συμβουλές ενός έμπιστου φίλου για να καθορίσετε μία λογική πορεία δράσης.
